- What command lists all branches in your repository, including both local and remote tracking branches?
git branch -a

- What does `remotes/origin/HEAD -> origin/main` mean?
This means that 'origin/main' is set as the default branch of the remote repository

- How is it different from `remotes/origin/main` and just `main`? 
'main' is the local branch where work is typically done
'remotes/origin/main' is the remote tracking branch that shows the state of the main branch on the remote repository
'remotes/origin/HEAD' is a pointer to the default branch on the remote repository
